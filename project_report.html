1. Abstract

CulinaryOS is a modern kitchen management system designed to solve common household problems related to food organization. By digitizing the pantry, simplifying recipe management, and enabling visual meal planning, CulinaryOS helps users save time, reduce waste, and improve their cooking experience. This report details the design, architecture, and implementation of the system.

2. System Overview

The application is built as a Single Page Application (SPA) using the MERN stack (MongoDB, Express, React, Node.js). It features a responsive design, real-time feedback, and secure authentication.

3. System Architecture

The system follows a standard Client-Server architecture. The React frontend communicates with the Express backend via RESTful APIs. Data is stored in a MongoDB database.

5. Database Design

We used Mongoose to model our data. Key schemas include User, PantryItem, and Recipe.

User Schema

Stores username, email, and hashed password.

PantryItem Schema

Fields: name, category, quantity, unit, user (ref).

6. Conclusion

CulinaryOS successfully addresses the need for a digital kitchen assistant. The modular design allows for easy scalability. Future improvements could include AI-powered recipe suggestions based on pantry expiry dates and mobile app development.
